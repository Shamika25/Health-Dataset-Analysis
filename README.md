# Health Data Analysis Using NumPy

## Project Overview

This project analyzes the NHANES (National Health and Nutrition Examination Survey) dataset using Python and NumPy. The objective is to perform statistical analysis, visualize body measurements, compute health indicators, and compare male and female participants based on various body measurements.


## Dataset

The project uses the following datasets:

- nhanes_adult_male_bmx_2020.csv
- nhanes_adult_female_bmx_2020.csv

These datasets contain body measurements of adult male and female participants.

---

## Features

- Read datasets as NumPy matrices
- Histogram comparison of male and female weights
- Box-and-whisker plots
- Statistical analysis (Mean, Median, Standard Deviation, Variance, Skewness, Kurtosis)
- BMI calculation
- Data standardization using Z-score
- Scatterplot matrix (Pairplot)
- Pearson and Spearman correlation analysis
- Waist-to-Height Ratio calculation
- Waist-to-Hip Ratio calculation
- Comparison using box plots
- Analysis of lowest and highest BMI participants
